# Portfolio Project Constitution

## Project Identity
**Name:** Huma Mohsin - Professional Portfolio
**Type:** Personal Portfolio Website
**Stack:** Next.js 15 + TypeScript + Tailwind CSS + Framer Motion

## Core Principles
1. **Performance First** - Fast loading, optimized assets
2. **Professional Design** - Inspired by high-end e-commerce aesthetics
3. **Subtle Animations** - Light AI-inspired micro-interactions
4. **Mobile Responsive** - Perfect on all devices
5. **SEO Optimized** - Discoverable by recruiters

## Design Reference
- Inspiration: Dribbble Sehlvet E-commerce design
- Color Palette: Elegant, professional tones
- Typography: Clean, modern hierarchy
- Spacing: Generous whitespace

## Sections
1. Hero (with profile picture)
2. About/Summary
3. Experience
4. Projects (with live links)
5. Skills
6. Education
7. Achievements
8. Contact

## Tech Decisions
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion (light, performant)
- **Icons:** Lucide React
- **Deployment:** Vercel
